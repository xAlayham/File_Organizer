# File Organizer Pro

A professional Python application that automates common file management tasks.

## Features

- Batch file renaming
- File categorization
- Duplicate detection
- Safe preview before changes
- Undo functionality
- Report generation
- Action logging
- Configurable settings

## Technologies

- Python
- Git
- Modular architecture
- File I/O
- Object-oriented design (later)
- Algorithms & data structures
- Automated testing
